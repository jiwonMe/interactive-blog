# useState 완전 정복

React Hooks 시리즈의 첫 번째 글입니다. `useState`에 대해 알아봅니다.

## 기본 사용법

```javascript
const [count, setCount] = useState(0);
```

useState는 상태 값과 그 상태를 갱신하는 함수를 반환합니다.
